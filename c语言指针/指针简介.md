## 数组
1. 计算机中数组存放在内存中是连续的 从低到高依次存放
2. 数据 列如int 在内存中占4位字节 按照小端原则 尾巴在前面 也是存在一篇连续的空间
3. 指针存放的是对应数据的首地址  操作系统的数据总线是多少位那么他就是多少位
4. 多维数组在内存中的存放也是连续的，随着下标的增加而增加。
5. 数组在定义时就要确定其大小
## 数组与指针
1. 数组是内存区间内紧密相连同种类型数据的集合，其数组名即为指向该数据类型的指针。

### 数组指针与函数指针
- 数组指针： int (\*p) [3]  由于()的优先级 标量p先于\*结合，表示它是一个指针 p的类型是 int (\*)[3]  赋值 int* arr1[3]={1,2,3};  \*p= arr1 p=&arr1
- 函数指针： int (\*p)(int a ,int b) 定义了一个指针变量p 它指向了 int num(int a , int b) 一个有两个int形参 一个int类型返回值的函数 p的类型是int (\*)(int x ,int y) 赋值 p = num 或p =&num
- 函数指针常用于回调函数，在特定的事件发生时，调用不同的函数。
