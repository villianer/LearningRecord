### C语言宏
1. #define 定义的  #define 宏名  替换内容
2. 预处理会直接将宏名替换为后面的内容
3. 无参宏 #define M (x+4)
4. 带参宏 #define #define M(x,y) ((x>y)?x:y)
	- 在带参宏定义中，形式参数不分配内存单元，因此不必做类型定义，这与函数中的情况是完全不同的，宏使用时只是将参数符号进行替换，不存在值传递的问题，形参根本不会分配空间。  
5. 使用宏定义一个代码块
	``` c
	#define Max(type,x,y) ({
	
	    type _x = x;
	
	    type _y = y:
	
	    (_x)>(_y)?(_x):(_y);
	
	})
	```
